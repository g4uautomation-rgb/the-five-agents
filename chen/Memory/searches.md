# זיכרון חיפושים — חן

לוג כל החיפושים של חן. **לפני** כל חיפוש חדש, חן עושה Grep על מילות
המפתח כאן כדי לבדוק אם חיפשה משהו דומה ב-30 הימים האחרונים. **אחרי**
כל חיפוש, חן מוסיפה entry בפורמט הקבוע למטה.

תבנית ל-entry:

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
